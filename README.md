## Deploying

Clone repo:

```
git clone https://github.com/estidlore/fair-pay.git
```

Run locally (inside project's root folder):

```
npm run start
```

Build (inside project's root folder):

```
npm run build
```

Deploy (if you're allowed to):

```
firebase deploy
```
